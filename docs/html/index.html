ToxiChemPy: Computational Framework for Toxicology ResearchÔÉÅ

ToxiChemPy is an open-source Python library for computational and experimental toxicology. It provides tools for cheminformatics, machine learning, statistical analysis, risk assessment, and data visualization, enabling researchers to analyze and predict the toxicological properties of chemicals.

The library is structured as a Python package (toxichempy), containing multiple subpackages that support data collection, toxicity modeling, and workflow automation.
